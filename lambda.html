Lambda-11 использует способность призывать мечи из ниоткуда для атаки врагов на всех дистанциях. В сочетании с порталами, которые также генерируют клинки, Lambda создаёт несколько источников урона, играя безопасно издалека. Несмотря на свою специализацию на дальней дистанции, она также превосходна в ближнем бою благодаря множеству мульти-хитовых движений, которые оглушают и связывают врагов.

Terrain Advantage
При падении врага с определённой высоты восстанавливает 60 MP. Высота не очень большая, но враг должен достичь достаточной скорости падения. Прерывание падения атакой отменяет восстановление MP. Множество синих сфер при падении указывает на успешное восстановление.
02

Навыки и приёмы

🌀
Luminous Slave EX
Legacy Skill 1
Открывает пространственные врата, выпускающие лезвия, которые автоматически нацеливаются на врагов. Частота появления лезвий увеличивается с каждым уровнем. Хорошее отслеживание и достойный урон.
🌊
Legacy Edge [EX]
Legacy Skill 2
Призывает гигантские пространственные врата, которые выпускают постоянный поток лезвий вперёд. Длительность увеличивается с каждым уровнем. Наносит урон долгое время на очень большой горизонтальной дистанции. Отлично против множества мелких врагов или боссов в углу.
🔫
[Ground] Attack — 2-hit Combo
Базовая атака
Комбо из 2 ударов: Lambda стреляет несколькими мини-лезвиями вперёд, затем обрушивает несколько вниз перед собой. С потенциалами: мини-лезвия автоматически нацеливаются на врагов выше; продолжают стрелять даже после прерывания; удержание направления и нажатие атаки заставляет Lambda непрерывно стрелять.
🕊️
[Airborne] Attack & Exiga Nail
Воздушная атака
Лезвия Lambda превращаются в большой серповидный клинок, которым она рубит вниз. Exiga Nail: [Airborne] Attack > Attack или [Airborne] Down + Attack — Lambda стреляет большим лезвием по диагонали вниз.
💨
Dash & Teleport
Уникальный Dash
Lambda ненадолго исчезает, телепортируясь вперёд. Имеет Dash Invulnerability. Dash > Attack: Lambda делает рывок вперёд и призывает большое лезвие, которое пронзает по диагонали вверх. С потенциалами: Triple Dash; лезвия рубят вокруг Lambda во время Dash; в воздухе Dash сбрасывает лимит [Airborne] Attack.
🗡️
Skill: 3 Waves of Giant Blades
Skill > Skill > Skill
Lambda призывает 3 волны гигантских лезвий (MP:33 каждая). С потенциалами: лезвия появляются в месте нахождения врага; Skill > Attack после любой из волн призывает дополнительную волну сверху (не тратит MP); больше лезвий в каждой волне.
⬆️
Heavy Strike
[Ground] Hold Attack
Lambda стреляет большим лезвием вперёд, слегка отступая. Имеет краткую неуязвимость. С потенциалами: при попадании несколько дополнительных лезвий атакуют врага; Dash Invulnerability во время использования; с Corsair Sabre — накладывает метку на врагов, которая призывает ещё больше лезвий.
⚔️
Sword Summoner EX
[Ground] Up + Attack x4 / [Airborne] Up + Attack x3
Lambda непрерывно призывает лезвия перед собой, атакующие врагов (до 4 раз на земле, до 3 раз в воздухе). Нельзя продолжить, если первый удар не попал. С потенциалами: ещё больше лезвий; во время атаки можно нажать Down + Attack для выстрела Exiga Nail в местонахождение врага.
🏹
Act Parcer Tri: Cavalier
[Ground] Down + Dash
Lambda выполняет плечевой таран и призывает большое лезвие из земли, которое разрушает Super Armor. С потенциалами: прерывание не останавливает призыв лезвия; можно использовать в воздухе; Dash Attack разрушает Super Armor.
⬆️
Up + Dash: Marking
Up + Dash
Lambda делает рывок вперёд, атакуя врагов по пути, и накладывает метку на одного поражённого врага (MP:50). Помеченный враг становится целью для автоматически призываемых лезвий. С потенциалами: ещё больше лезвий; до двух врагов могут быть помечены.
⬇️
Down + Skill: Massive Blade
Down + Skill / Down + Hold Skill
Призывает массивное лезвие с неба, которое падает на землю (MP:100). Down + Hold Skill: призывает 3 массивных лезвия вместо одного (MP:100). Отличный инструмент для нанесения урона по области и контроля зоны.
🛡️
Triple Jump & Air Reset
Потенциал
Lambda может совершать тройной прыжок. При получении урона может немедленно восстановиться прыжком и стать неуязвимой на 1 секунду. В воздухе может использовать Jump для сброса лимита [Airborne] Attack. С Luminous Slave EX: прыжок в воздухе порождает маленький портал, который стреляет дополнительным лезвием.
03

Основные механики

Sword Summoner — множество источников урона
Lambda-11 специализируется на призыве лезвий из порталов. Её навыки создают несколько источников урона одновременно, позволяя ей играть безопасно издалека. Luminous Slave EX автоматически нацеливается на врагов, Legacy Edge создаёт постоянный поток лезвий, а Sword Summoner EX позволяет непрерывно атаковать врагов перед собой. В сочетании с возможностью телепортации и тройным прыжком, Lambda может постоянно перемещаться и наносить урон, оставаясь трудной мишенью.
🎯
Автоматическое нацеливание
Luminous Slave EX и помеченные враги (через Up + Dash) позволяют Lambda наносить урон, не прицеливаясь вручную. Это даёт ей свободу перемещения и уклонения.
💨
Телепортация и мобильность
Dash Lambda — это телепорт с полной неуязвимостью. В воздухе Dash сбрасывает лимит воздушных атак. С Triple Dash она становится чрезвычайно мобильной.
🔄
Восстановление MP
Terrain Advantage даёт 60 MP при падении врага с высоты. Хотя это не должно быть главным приоритетом, это полезный бонус для поддержания запаса MP.
⚡
Over-Exhaust MP
Универсальные навыки Lambda позволяют ей тратить MP сверх лимита (Over-Exhaust). Это даёт возможность использовать больше навыков в критических ситуациях.
04

Советы по игре

  • 🌀Используй Luminous Slave EX для пассивного урона. Порталы автоматически нацеливаются на врагов, позволяя тебе сосредоточиться на уклонении и позиционировании.
  • 🌀Legacy Edge отлично подходит для боссов в углу. Постоянный поток лезвий наносит огромный урон, если босс заперт у стены.
  • 🌀Освой телепортацию. Dash Lambda даёт полную неуязвимость. Используй его для уклонения от опасных атак и быстрого перемещения.
  • 🌀Сбрасывай лимит воздушных атак. В воздухе Dash или Jump сбрасывают лимит [Airborne] Attack, позволяя выполнять бесконечные воздушные комбо.
  • 🌀Sword Summoner EX — отличный инструмент для ближнего боя. Непрерывный призыв лезвий оглушает врагов и наносит высокий урон.
  • 🌀Помечай приоритетные цели. Up + Dash накладывает метку, которая призывает дополнительные лезвия. Идеально для боссов и элитных врагов.
  • 🌀Down + Hold Skill для мгновенного AoE. Три массивных лезвия с неба наносят огромный урон по области — отлично для зачистки групп.
